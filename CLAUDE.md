# CLAUDE.md

Guidance for working in this repo. See [ARCHITECTURE.md](ARCHITECTURE.md) for the system's shape.

## Maintaining ARCHITECTURE.md

Whenever you add, remove, or significantly change a feature, module, data model, or architectural pattern, update ARCHITECTURE.md in the same session to reflect the change. Keep edits surgical — update only the affected sections. If a change is purely cosmetic or a minor bug fix, leave the doc alone.
